# Cyber Security Roadmap - Progress Tracker

A comprehensive, interactive progress tracker for a structured 26-week cybersecurity learning journey. Designed specifically for web developers transitioning into professional cybersecurity roles.

## 🎯 Overview

This project provides a professional roadmap covering 11 phases of cybersecurity fundamentals, from computer architecture to practical penetration testing. Track your progress, take notes, and build an audit-ready skill set step by step.

**Duration:** 26 weeks | **Topics:** 200+ core concepts | **Phases:** 11 structured modules

## ✨ Features

- **Structured Learning Path** - 11 comprehensive phases with clear objectives and outcomes
- **Progress Tracking** - Real-time progress visualization with circular progress indicators
- **Phase-Based Learning** - Organized topics, practical exercises, and learning outcomes
- **Search & Filter** - Quickly find topics by keyword or filter by progress status
- **Notes Management** - Save personal notes for each topic during learning
- **Export/Import Progress** - Backup and restore your learning progress as JSON
- **PDF Export** - Print or save your roadmap and progress as a PDF document
- **Local Storage** - All progress is saved locally in your browser
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Dark Theme** - Professional dark interface optimized for extended study sessions

## ⚙️ Troubleshooting

**Progress not saving?**
- Check if Local Storage is enabled in your browser
- Ensure cookies/storage aren't being cleared on browser close

**Can't import progress?**
- Verify the JSON file is valid and from this application
- Try exporting current progress first to check format

**Display issues?**
- Refresh the page (Ctrl+R or Cmd+R)
- Clear browser cache if styles seem broken
- Try a different browser if problems persist
